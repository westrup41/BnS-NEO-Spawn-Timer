# B&S NEO Spawn Timer v4.1

Таймеры Императорского древа, мировых боссов и пользовательских Event,
оверлей, зашифрованный онлайн-чат и P2P-алерты через бесплатные MQTT-узлы.

В v4.1 добавлены приватные комнаты, защита чата, подписанная модерация,
диагностика сети, импорт/экспорт, резервные копии и проверяемое автообновление.

В настройках интерфейса доступны темы `Classic`, `Midnight` и светлая `Starlight`.
